## Installation
Use docker to build and run the flask app

1. cd into the ```app``` dir
2. Build
``` 
docker build -t coolvision-app .
```

3. Run
``` 
docker run -p 5000:5000 coolvision-app
```

4. open http://172.21.80.246:5000
